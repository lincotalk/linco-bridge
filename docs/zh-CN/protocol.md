# Protocol 公开协议

[English](../protocol.md)

本文档说明 CLI、Server、Reference Web 与兼容客户端共用的公开契约。发布前必须基于已审计实现完成。

## 必备内容

1. 协议版本和兼容性协商。
2. 设备注册和凭证校验。
3. 心跳和在线 / 离线状态。
4. Agent、project、profile 和 session 标识。
5. 会话索引同步和消息分页。
6. 消息、工具、权限、文件和错误事件。
7. 重连、去重、顺序和超时行为。
8. 凭证过期、撤销、刷新和占用处理。

不在公开文档中提供生产环境内部端点、签名私钥和风控规则。
