# Reference Web

[English](../reference-web.md)

Reference Web 是用于验证 Linco Connect 链路的开放客户端。其响应式移动布局是 H5 体验，不等同于微信小程序。

## 包含

- 选择已支持 Agent。
- 创建或输入连接凭证。
- 复制安装和启动命令。
- 检测连接状态。
- 同步、浏览和搜索会话索引。
- 进入 session 并进行受限测试。
- 展示数据路径、隐私和产品状态说明。

## 不包含

- Linco App 完整 IM 和协作能力。
- 生产级多设备管理。
- 生产可用的自托管控制面。
- 未公布真实链接的官方托管服务。
