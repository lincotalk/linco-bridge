## 发布规则

- 保持 README、Release Notes 和兼容性说明的一致。
- 完成实际验证后，再在 Release 中公布精确测试版本。
- 如果某个 Agent 或操作系统存在限制，请在 Release Notes 和排障文档中明确写出，而不是让发布范围保持模糊。
